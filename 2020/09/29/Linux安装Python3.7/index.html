Linux安装Python3.7

环境: CentOS 6.6

在Linux CentOS 6.6测试环境上安装Python3.9环境，折腾了一天，各种找文档，还是各种报错，奔溃但最终还是搞定了~

本文档将记录我的环境可以快速成功安装相关步骤，如果您遇到其他问题，也可以参考最后几个文档，顺藤摸瓜。

问题背景

  1. Python官网 下载Python3.7.9源码包，’./configure; make; make install’之后，使用pip安装包全部报错 Can”t connect to HTTPS URL because the SSL module is not available；

  2. 安装openssl1.1之后，参考Building Python 3.7.1 - SSL module failed重新make&make install 还是报错：

安装Python3.7.9

安装步骤

  1. 下载Python3.7.9：https://www.python.org/ftp/python/3.7.9/Python-3.7.9.tgz

  2. ./configure –with-openssl

  3. 修改 Modules/Setup
    a. 解除注释 _socket socketmodule.c
    b. 解除注释SSL相关配置，SSL=/usr/local/ssl，路径是上面libressl的安装路径

  4. make & make install，如果需要查看详细输出/错误信息，可以将错误输出、标准输出重定向到文件。

    make步骤的标准输出中，如果看到ssl模块已经build成功，说明make成功了。

  5. 最后是 make install
