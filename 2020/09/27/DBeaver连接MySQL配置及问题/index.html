DBeaver连接MySQL配置及问题

DBeaver配置MySQL连接

基础设置

  1. 新建->DBeaver->数据库连接->筛选及选中MySQL->创建新连接窗口
  2. Server：服务器地址默认localhost，端口默认3306
  3. 认证：用户名root，密码是安装时设置的root用密码

编辑驱动设置

  1. 库：删除默认的jdbc连接组件–自动下载太慢。
  2. 下载jdbc-connector：https://dev.mysql.com/downloads/ 选择Connector/J，选择 Platform Independent。下载压缩文件，解压到MySQL同级目录下，比如：C:\Program Files\MySQL。
  3. 回到驱动设置-库，选择文件，文件为上述步骤中下载的jar包文件，比如：C:\Program Files\MySQL\mysql-connector-java-8.0.21\mysql-connector-java-8.0.21\mysql-connector-java-8.0.21.jar
  4. 本地客户端：选择MySQL Server 8.0 或者你自己的MySQL服务

1. mysql连接失败: The server time zone value is unrecognized

解决方案参考：https://www.cnblogs.com/godwithus/p/9788790.html

我采用的是：修改MySQL数据库配置，使用root用户登录mysql
