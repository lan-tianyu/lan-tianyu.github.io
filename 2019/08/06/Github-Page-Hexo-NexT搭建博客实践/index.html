Github Page 配置

  1. 注册一个Github账号（已经注册过）
  2. 创建一个同名仓库(create a repository)的代码托管一个静态站点：xxx.github.io。一定是与Github账号同名，否则打开是404的。
  3. 配置ssh key（已经配置过）
  4. 部署到Github.
